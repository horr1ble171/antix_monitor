# [AntiX Monitor](https://antix-monitor.vercel.app/)

Простой и легковесный веб-монитор для вашей системы, разработанный на Python и HTML/CSS/JS. Предоставляет актуальную информацию о состоянии CPU, RAM, диска, сети и запущенных процессах в удобном веб-интерфейсе. Идеально подходит для мониторинга серверов или одноплатных компьютеров (например, Raspberry Pi) под управлением Linux.

## ✨ Особенности

*   **Мониторинг в реальном времени:** Получайте актуальные данные о вашей системе каждые несколько секунд.
*   **Полный набор метрик:** Отслеживайте загрузку CPU, использование оперативной памяти, занятое пространство на диске, сетевую активность (отправлено/получено), температуру, аптайм и количество процессов.
*   **Топ процессов:** Просматривайте список наиболее ресурсоемких процессов с информацией о PID, имени, загрузке CPU и использовании RAM.
*   **Легковесный:** Минимальные зависимости, работает "из коробки" с Python 3.
*   **Веб-интерфейс:** Доступ к монитору через любой современный веб-браузер.
*   **Простая установка:** Запустите сервер одной командой.
*   **Современный дизайн:** Чистый и адаптивный интерфейс, удобный для просмотра на разных устройствах.

## 🛠️ Технологии

*   **Backend:** Python 3 (встроенный HTTP-сервер)
*   **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)

## 🚀 Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/horr1ble171/antix_monitor.git
    cd antix_monitor
    ```

2.  **Запустите сервер:**
    ```bash
    python3 server_simple.py
    ```
    Сервер будет запущен на порту `8080`. Вы увидите сообщение: `🚀 Монитор запущен на http://0.0.0.0:8080`

## 🌐 Использование

Откройте ваш веб-браузер и перейдите по адресу: `http://localhost:8080` (или `http://АДРЕС_ВАШЕГО_СЕРВЕРА:8080`, если вы запускаете его на удаленной машине).

Интерфейс будет автоматически обновляться, показывая текущее состояние вашей системы.
